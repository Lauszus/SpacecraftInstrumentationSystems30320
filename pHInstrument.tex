* pH
	* Solid state (ISFET)
		* No calibration
		* Measures current flow at different voltages, H+ ions
	* Temperature compensated [5:50] degrees C
		* Response time ~60 s
	* 200 uL, ~20 mW, below 10 mA
	* $\pm$ 0.01 pH
* Salinity can use same instruments using membranes
	* Current flow
* Temperature probe
	* K-type thermal couple
